# CIIUTI：男团角色测试（初版设计）

这个文档是你要的“可复刻 SBTI、但更科学且更有传播性”的第一版落地方案。

## 1) 目标定义

- **产品目标**：3-5 分钟完成测试，得到“你是男团 5 人 + 老板 JYP 的谁”共 6 个结果。
- **双重属性**：
  - 科学依据：用可解释的心理维度 + 反向计分 + 置信度机制。
  - 传播属性：结果文案“半认真半整活”，支持晒图、对比、挑战链路。

## 2) 参考 SBTI 的可复用点与改造点

### 可复用
- 题量短（31 题左右）+ 低门槛（A/B/C）提高完测率。
- 结果命名强记忆点，文案有“梗感”。
- 距离匹配模型（用户向量 vs 角色向量）简单好维护。

### 需要改造
- 每维度 2 题过少，稳定性不足。
- 缺少反向计分，维度一致性容易失真。
- 缺少质量控制（极端作答、一条线作答、随机作答识别）。

## 3) 推荐测试结构（比 SBTI 题更少）

- **题量建议**：18 题（正式）+ 2 题可选彩蛋题。
- **维度数量**：6 个核心维度，每维 3 题（其中至少 1 题反向计分）。
- **选项数量**：5 点李克特（非常不同意 ~ 非常同意）。
- **结果数量**：6 个（成员 A/B/C/D/E + JYP）。

> 这样比 SBTI 更短，同时比“2题一维”更稳。

## 4) 六维模型（可解释 + 贴合男团语境）

- **D1 舞台主导（Stage Dominance）**：存在感、控场、镜头中心倾向。
- **D2 团队协同（Team Harmony）**：配合、补位、关系维护。
- **D3 自律打磨（Discipline & Grind）**：训练强度、细节要求、持续性。
- **D4 创作表达（Creative Voice）**：审美表达、创意输出、个人风格。
- **D5 情绪张力（Emotional Intensity）**：感染力、情绪外显、叙事张力。
- **D6 商业/掌控（Executive Control）**：资源调度、策略思维、结果导向。

## 5) 评分机制（科学与传播兼顾）

### 5.1 题目计分
- 每题 1~5 分。
- 反向题使用 `score = 6 - rawScore`。

### 5.2 维度得分
- 每维取 3 题均值，归一化到 `0~100`。
- `dimScore = ((mean - 1) / 4) * 100`。

### 5.3 结果匹配
- 每个角色配置 6 维原型向量（0~100）。
- 用**加权曼哈顿距离**或**加权欧氏距离**计算相似度。
- 推荐公式：
  - `distance = Σ (weight_i * |user_i - proto_i|)`
  - `similarity = max(0, 100 - distance / maxDistance * 100)`

### 5.4 置信度与防刷
- **置信度**：第一名和第二名差值 `< 6%` 时显示“双人格边界提示”。
- **作答质量控制**（可选）：
  - 连续同一选项占比过高（例如 >80%）标记低质量。
  - 极短作答时长提示“建议重测”。

## 6) 传播机制建议

- **结果文案结构**：`角色名 + 口号 + 夸张梗 + 真正建议`。
- **挑战机制**：一键生成“我是谁，你敢来测吗”分享图。
- **对比机制**：可输入好友结果，生成“同团搭子指数”。
- **彩蛋机制**：2 道非计分题触发隐藏文案（不改变主结果，防破坏测量）。

## 7) 文件说明

- `config/schema.json`：配置结构定义。
- `config/test-config.example.json`：你要填写的主配置模板（题库先留空）。
- `docs/content-writing-guide.md`：你后续填写题目和结果文案的规范。
